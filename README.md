# mysql-food-ordering
This project is designed using MySQL to store, manage, and retrieve data efficiently. It includes database tables, relationships, constraints, and sample queries to demonstrate real-time database operations. The project focuses on data entry, updating, deletion, and reporting using SQL commands.



Key Features:

Proper table structure with primary & foreign keys

CRUD operations (Create, Read, Update, Delete)

Joins, Subqueries, Group Functions

Data integrity and constraints
